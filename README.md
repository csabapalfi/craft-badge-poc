# craft-badge-poc

Gated proof-of-concept for a privacy issue I found in my own event ticket and
reported privately to the organisers.

`index.html` is a camera QR scanner that is **deliberately gated**: it only opens
the author's own demo record and silently ignores any other code, so it cannot be
used against anyone else.

Technical details withheld here until the affected parties have had a chance to
address it.
