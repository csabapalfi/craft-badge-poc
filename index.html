What this shows: the badge QR is the bare Tito ticket ID. The ID maps to https://ti.to/tickets/<id>.json, which returns the attendee's full record (name, employer, work + personal email, mobile, demographics) with no login.

Gated on purpose: this public page only reveals data for the author's own demo badge and refuses every other ID, so it cannot be used against other attendees. Found on my own ticket; reported privately to the organisers.
